Navbar mobile toggle

Add the toggles to the navbar mobile toggle, to show and hide the menu.

All you need to do is add data-bulma="navbar" to the main navbar div, and then add data-trigger and data-target="navID" to the navbar burger.

Resize your browser to test the mobile nav.

DOM options

Option name Required Default Type Description
data-bulma Yes Nothing String This is the component type, to enable the navbar functionality this should be navbar.
data-trigger Yes Nothing String The element you would like to be used as the trigger. This will assign the click event listener.
data-target Yes Nothing String This is applied to the same element the data-trigger attribute is applied to. This is a string containing the ID of the navigation menu.
