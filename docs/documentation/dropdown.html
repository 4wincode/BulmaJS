Dropdown

{% include notice.html %}

Dropdown toggle

Adds the toggle functionality to a dropdown component. If you are using is-hover this is not required.

To add the JS magic to a dropdown component, add a data-bulma="dropdown" attribute to the main dropdown element. Also add data-trigger to the button element. The rest will be taken care off behind the scenes.

DOM options

Option name Required Default Type Description
data-bulma Yes Nothing String This is the component type, to enable the dropdown functionality this should be dropdown.
data-trigger Yes Nothing String The element you would like to be used as the trigger. This will assign the click event listener.
